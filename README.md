# uncrustify_config
Prototype Uncrustify web configurator based on the emscripten interface of Uncrustify.

----------------------

| Instructions 	|   	|
|--------------	|---	|
| Build:       	| `npm install` 	|
| Run:         	| open index.html with your webbrowser 	|
| Preview:     	| https://cdanu.github.io/uncrustify_config_preview/index.html 	|
